# 2UHRFERTIG

Ein interaktives Spiel-Projekt mit Node.js Backend und Browser-Frontend.

## Projektstruktur

- `src/` - Client-side JavaScript Module
- `public/` - Statische Assets (HTML, CSS, Sounds)
- `server.js` - Node.js Server
- `accounts.json` - Spieler-Accounts
- `leaderboard.json` - Bestenliste

## Installation

```bash
npm install
npm start
```

Das Spiel läuft dann auf http://localhost:3000
